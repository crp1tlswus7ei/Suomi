# Suomi - Build Your Own Bot

Welcome! Suomi is an open-source bot designed for moderation, automation, and advanced server management.
If you'd like to use the code to build your own bot, [here](https://discord.gg/KEfpB6yDJN) is a 
tutorial, and you'll also join the Suomi support community! Thank you for supporting this project.

# Running setup script

An installation script has been implemented to simplify the installation process.
It may contain some errors when run, so it is best to run it with administrator privileges in Windows.

If you encounter any problems, you can join the support server and post a comment in the support channel.
